# Willy Wonka’s World 🍫✨

## Overview
Welcome to **Willy Wonka’s World**, an interactive front-end demo website inspired by the magical world of Willy Wonka and his famous chocolate factory. This website showcases a variety of features including detailed pages about Willy Wonka’s character, his iconic factory, his films, and a shop where visitors can “buy” magical products. The site is designed to be a fun and engaging experience, while also demonstrating modern front-end web technologies.

This project demonstrates various web concepts such as responsive design, DOM manipulation, jQuery interactions, multi-step forms, and interactive UI components. It provides a playful yet polished look into Willy Wonka’s universe, complete with movie galleries, chocolate factory details, fun facts, and a simulated shopping experience.

---

## Pages Included

### 1. **Home Page (willy.html)**
   - The homepage introduces visitors to Willy Wonka's magical world. It features a hero section, floating animations (bubbles, sparkles), and a dynamic greeting message.
   - It also includes a variety of interactive components such as:
     - **Theme Toggle**: Switch between light and dark mode.
     - **Time Display**: Real-time clock.
     - **Background Color Change**: Change the page’s background color with a button.
     - **Magical Visitors Counter**: An animated counter for visitors.
     - **Multi-Step Golden Ticket Form**: A fun form where users can join the factory crew.
     - **Interactive Rating & Random Facts**: Rate Willy Wonka's World and get random facts about Willy Wonka’s story.

### 2. **About Page (index.html)**
   - A detailed introduction to Willy Wonka’s character, the chocolate factory, and key items like the **Everlasting Gobstopper** and **Chocolate River**.
   - Features a **lazy-loaded gallery** of images from the movies, with descriptions about various depictions of Willy Wonka through the years.
   - Includes an interactive **FAQ Section**, a **Contact Form** to send a "Golden Ticket", and a **Multi-Step Join the Factory Crew Form**.

### 3. **Movies Page (movies.html)**
   - Explores the different film adaptations of Willy Wonka over the years, with special focus on:
     - The **1971 Gene Wilder** film, **2005 Johnny Depp** movie, and the **2023 Timothée Chalamet** prequel.
   - Provides movie cards with images, descriptions, and a button to "Learn More".
   - Includes a **searchable gallery**, a **FAQ Section** about the films, and an interactive **rating system**.
   - Features a **multi-step Golden Ticket application form**.

### 4. **Shop Page (shop.html)**
   - A simulated online shop where users can explore magical products, such as Willy Wonka’s signature chocolates and candies.
   - **Product Cards**: Display products with images, descriptions, and prices.
   - **Category Filter**: Users can filter products by category (e.g., chocolate, candy).
   - **Multi-step Checkout Form**: Simulated form for checking out products.
   - A **gallery** showcasing various sweet treats, and a **theme toggle** to change the background.

## Features
- **Responsive Layout**: All pages are mobile-friendly, using a flexible grid layout for seamless viewing across devices.
- **Interactive UI**: Includes buttons, sliders, forms, modals, and star ratings.
- **jQuery Integration**: Utilizes jQuery for DOM manipulation, event handling, and animations.
- **Form Validation**: Multi-step forms with proper input validation for the Golden Ticket application and contact forms.
- **Pop-up Modals**: Used for subscriptions and product information.
- **Lazy Loading**: Optimized image loading in the gallery for better performance.

---

## Tech Stack
- **HTML5**: Semantic and structured content.
- **CSS3**: Used for layout, styling, and animations.
- **JavaScript (ES6)**: Used for logic and interactive features.
- **jQuery**: For DOM manipulation and event handling.
- **External CDN**: jQuery 3.7.1 for easy integration.
- **No Backend**: All forms are demo only, and no server-side processing is implemented.

---

## Features Breakdown
- **Theme Toggle**: Switch between light and dark modes for a better user experience.
- **Current Time Display**: Shows real-time updates of the system's current time.
- **Searchable Galleries**: Users can search for specific movies, products, or characters.
- **Interactive Forms**: Multi-step forms guide users through the Golden Ticket registration process.
- **Pop-up Subscription**: A modal for users to subscribe to Willy Wonka's updates.

## How to Use
1. **Download/Clone** the project folder.
2. Ensure all files are in the same directory, maintaining the structure of HTML, CSS, JS, and images.
3. Open **willy.html** in a browser to view the main page.
4. Use the navigation bar to move between the **About**, **Movies**, and **Shop** sections.
5. Explore interactive elements like the **theme toggle**, **background color change**, and **multi-step forms**.

> Note: This is a static front-end project, and no server-side functionality is needed to run it.
